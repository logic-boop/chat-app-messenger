**Vesta Messenger** is a high-performance, responsive communication platform engineered by **Vesta Systems**. This project is a **Minimum Viable Product (MVP)** designed to demonstrate the power of modern serverless architecture, real-time data synchronization, and premium UI/UX design.

## 🚀 Vision
The goal of Vesta Messenger is to bridge the gap between high-ticket aesthetic and functional utility. It provides a seamless, cross-platform chat experience with features inspired by industry leaders like Facebook Messenger.

## ✨ Key Features (MVP)
- **Premium Landing Page:** A professional entry point for users.
- **Enterprise-Grade Auth:** Secure login/signup with persistent sessions.
- **Real-time Synchronization:** Instant message delivery and typing indicators.
- **Adaptive UI:** Fully responsive design with native-feeling Light/Dark modes.
- **Productivity First:** Keyboard-driven interaction (Enter to send).

## 🛠️ Tech Stack
- **Frontend:** Next.js 15+ (App Router), Tailwind CSS.
- **Backend:** Next.js Server Actions & API Routes.
- **Real-time:** Pusher Channels (Vercel-optimized WebSockets).
- **Security:** Lucide-React for interface logic and custom Auth-guards.

## 📦 Installation
1. Clone the repo: `git clone https://github.com/your-username/vesta-messenger.git`
2. Install dependencies: `npm install`
3. Set up Environment Variables in `.env.local` (Pusher, Database keys).
4. Run locally: `npm run dev`
